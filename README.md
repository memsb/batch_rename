# Batch Rename

A tool for batch renaming files in a directory.

When run the tool asked for input and output directories followed by a formatting string.

| Character | Output |
----------- | ---- |
| %c | Creation Date | 
| %m | Last Modified Date | 
| %d | Current Date | 
| %b | Original Name | 
| %n | Sequence Number | 
| %e | File Extension |
